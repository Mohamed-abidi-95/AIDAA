# دليل الطلكنسولتاسيون — AIDAA 🎥
### كيفاش تستعمل feature الـ Téléconsultation بالتونسي 🇹🇳

---

## 🔎 شنوّا هي الـ Téléconsultation؟

الـ téléconsultation هي feature تخلّي الدكتور (professionnel de santé) يعمل **جلسة فيديو** مع المريض أو مع وليّ أمره مباشرةً من الكمبيوتر — بلا ما يتنقّل حتى واحد.

من خلالها تنجّم:
- 🗓️ **تحدّد موعد** جلسة مع patient محدد
- 📋 **تشوف لائحة** كل الجلسات المبرمجة
- 🎥 **تدخل قاعة الفيديو** وتبدا جلستك مباشرةً
- 💬 **تكتب في الشات** أثناء الجلسة

---

## 🚀 كيفاش تدخل على الـ Téléconsultation؟

### الطريقة 1 — من الـ Dashboard الرئيسي

1. ادخل على حسابك كـ **professionnel**
2. تلقى في الـ **sidebar** على اليسار بوتون **🎥 Téléconsultation**
3. اكبس عليه — يودّيك على صفحة **قائمة الجلسات**

```
URL: http://localhost:5173/professionnel/teleconsultation
```

### الطريقة 2 — مباشرةً من URL

```
http://localhost:5173/professionnel/teleconsultation
```

> ⚠️ **ملاحظة مهمة:** لازم تكون logged in كـ professionnel — كان ما كنتيش authenticated, يرجّعك للـ login page.

---

## 📋 صفحة قائمة الجلسات (TeleconsultationList)

```
URL: /professionnel/teleconsultation
```

هذي هي الصفحة الرئيسية للـ téléconsultation. فيها:

| العنصر | وصفو |
|--------|------|
| **جدول الجلسات** | يبيّن اسم المريض، التاريخ، الوقت، المدّة والحالة |
| **Badge الحالة** | 🔵 Planifiée = مبرمجة / 🟢 En cours = دايرة هاذم / ⚫ Terminée = خلصات |
| **بوتون "▶ Démarrer"** | للجلسات المبرمجة — يودّيك لقاعة الفيديو |
| **بوتون "🟢 Rejoindre"** | للجلسات الدايرة الآن — تدخل فيها مباشرةً |
| **بوتون "+ Planifier une session"** | تبرمج جلسة جديدة |

### شنوّا يعني كل badge؟

```
🔵 Planifiée  = الجلسة مبرمجة — مزالت ما بداتش
🟢 En cours   = الجلسة شغّالة الآن — تنجّم تدخل
⚫ Terminée   = الجلسة خلصات — تنجّم تشوف التفاصيل
```

---

## 🗓️ كيفاش تبرمج جلسة جديدة؟

1. اكبس على بوتون **"+ Planifier une session"** (بالكرتون برتقالي فوق اليمين)
2. تلقى فورم فيه:

```
┌─────────────────────────────────────────────┐
│  📋 Planifier une session                    │
├─────────────────────────────────────────────┤
│  Patient *      →  اختار من القائمة          │
│  Date *         →  حدد التاريخ               │
│  Heure *        →  حدد الوقت                 │
│  Durée          →  30 / 45 / 60 دقيقة        │
│  Notes          →  اختياري — ملاحظاتك        │
└─────────────────────────────────────────────┘
```

3. بعد ما تعمر كل شي، اكبس **"✅ Confirmer la session"**
4. يظهر لك بانر أخضر يقولك "Session planifiée avec succès!" ويرجّعك للقائمة

> 💡 **تيپ:** كان بدّيت تعدّل، اكبس **"Annuler"** تحت الـ form — يرجّعك للقائمة بلا ما يحفظ شي

## 🎥 قاعة الفيديو (TeleconsultationRoom)

```
URL: /professionnel/teleconsultation/[رقم-الجلسة]
مثال: /professionnel/teleconsultation/1
```

كيفاش تدخل:
- اكبس **"▶ Démarrer"** أو **"🟢 Rejoindre"** من قائمة الجلسات

### تصميم القاعة

```
┌──────────────────────────────┬──────────────────┐
│                              │  👤 معلومات       │
│    📷  Zone Vidéo             │     المريض        │
│    (مساحة الكاميرا)           ├──────────────────┤
│                              │  💬  Chat         │
│    🙂 self-view (صورتك)       │  ─────────────── │
│    (كوادري صغير في الركن)     │  [اكتب هنا...]   │
├────────────────── ───────────┴──────────────────┤
│  🎙️   📹   🖥️              ⬛ Terminer           │
│ (Micro)(Cam)(Partage d'écran)(إنهاء الجلسة)     │
└─────────────────────────────────────────────────┘
```

---

## 🎛️ شرح أزرار التحكم

| البوتون | الأيقونة | وظيفتو |
|---------|----------|---------|
| **Micro** | 🎙️ / 🔇 | تفعّل أو تقطع الميكروفون |
| **Caméra** | 📹 / 🚫 | تفعّل أو تطفي الكاميرا |
| **Partage d'écran** | 🖥️ | تشارك شاشتك مع المريض |
| **Terminer** | ⬛ (أحمر) | تنهي الجلسة وترجع للقائمة |

> ⚠️ **مهم:** بوتون **"Terminer"** باللون الأحمر — كي تكبسو يرجّعك فوراً للقائمة. الجلسة تعتبر خلصات.

## 💬 كيفاش تستعمل الـ Chat؟

الـ chat موجود في **اليمين** على جنب معلومات المريض:

1. اكتب رسالتك في مربع **"Saisissez un message…"**
2. اضغط **Enter** أو اكبس **بوتون →** (برتقالي)
3. تظهر رسالتك في الـ chat بتاريخ ووقت الإرسال

> 💡 **ملاحظة:** في النسخة الحالية الـ chat يحفظ الرسائل في الـ session (local state). لما تخرج من القاعة الرسائل تمحى — هذا طبيعي في النسخة الأولى.

## ❓ أسئلة شايعة (FAQ بالتونسي)

**سؤال: شنوّا يصير كي ما عنديش internet؟**
> الكاميرا تبقى "في الانتظار" — يظهر لك رسالة "Caméra en attente". WebRTC (الفيديو الحقيقي) سيُضاف في نسخة قادمة.

**سؤال: ينجّم يكون عندي أكثر من جلسة في نفس الوقت؟**
> في النسخة الحالية، تنجّم تشوف كل جلساتك في القائمة. كل جلسة لها ID خاص بيها.

**سؤال: كيفاش نضيف patient جديد؟**
> المرضى يُضافوا من **Dashboard الرئيسي** ← "Mes patients". ما ينجّمش تضيفهم من صفحة الـ téléconsultation.

**سؤال: الـ mock data هذي دائمة؟**
> لا. هذي بيانات تجريبية (4 جلسات). في الـ production، الجلسات تتحفظ في قاعدة البيانات عبر API حقيقية.

**سؤال: كيفاش نرجع للـ dashboard الرئيسي؟**
> اكبس على **"Mes patients"** أو أي بوتون آخر في الـ sidebar على اليسار — يرجّعك لـ `/professional/dashboard`.
